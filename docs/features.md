# Niubility 功能清单

> 最后更新：2026-04-11

本文档按“代码当前状态”整理功能，不把设计稿或未来规划写成已上线能力。

| 模块 | 当前状态 | 说明 |
|------|------|------|
| 认证与初始化 | ✅ 已完成 | 密码登录、注册开关、OIDC/SAML SSO、初始化超管 |

| 内容系统 | ✅ 已完成 | `video`、`gallery`、`article`、`podcast` |

| CLI | 🟡 部分完成 | 已可登录、浏览、发布文章和管理常见资源 |

## 认证与初始化

### 已实现

- 密码登录
- JWT Cookie 会话
- 注册开关控制
- OIDC SSO
- SAML 2.0 SSO
- 首次启动初始化超级管理员
- 首次启动自动生成 JWT 密钥与加密密钥
- CLI 浏览器 SSO 登录链路：`/api/v1/sso/cli/start`、`/api/v1/sso/cli/login`、`/api/v1/sso/cli/exchange`

### 说明

- `GET /api/v1/boot` 会返回初始化状态、认证状态、分类、注册开关、SSO 开关和站点配置
- 运行期认证配置保存在 `settings` 表中，不依赖第二套本地密钥配置

- 修改密码 / 判断是否已有密码
- 用户主页
- 关注 / 取消关注
- 关注列表 / 粉丝列表
- 收藏列表
- 我的内容
- 我的浏览记录
- 我的点赞

### 说明

- 用户主页已支持按内容类型切换，其中包括 `podcast`
- 关注动态页已上线

## 内容系统

### 内容类型

| 类型 | 状态 | 说明 |
|------|------|------|
| `video` | ✅ | 支持视频附件、主讲人信息、播放相关展示 |
| `gallery` | ✅ | 支持图片/短视频附件集合 |
| `article` | ✅ | 支持富文本正文与附件 |
| `podcast` | ✅ | 支持播客详情与编辑页，前后端类型已接通 |

### 已实现能力

- 创建、编辑、删除内容
- 草稿 / 发布状态
- 分类、标签、简介、封面
- 主讲人信息
- 附件上传、排序、封面标记
- 内容详情页分享能力
- 内容浏览计数、点赞、收藏、评论
- 后台按内容类型管理：`videos`、`galleries`、`articles`、`podcasts`

### 点赞

## 后台管理

### 已实现页面

- 内容管理
- 用户管理
- 分类管理
- 站点设置
- 认证设置
- 存储设置
- 企业微信设置
- 数据库备份
- 服务节点

### 说明

- 后台设置页已覆盖站点品牌、注册开关、Cookie 安全、OIDC、SAML、S3、资源分发、图片样式、企业微信配置等
- 管理后台默认入口是内容管理页

## 存储、资源分发与站点资源

### 已实现

- S3 兼容对象存储上传
- `presign -> 客户端直传 -> key 回写` 上传模式
- 头像、附件、站点资源三条访问链路
- 可配置公开访问 URL
- 可配置资源分发 provider
- 分发私有访问开关
- 分发 URL TTL
- 图片样式配置

### 当前已知 provider

- `disabled`
- `qiniu`

## 数据库备份

### 已实现

- 管理员手动触发数据库备份
- 备份历史查询
- 备份下载签名链接
- PostgreSQL / MySQL 双支持
- `.sql.gz` 压缩归档
- 备份文件上传到现有对象存储
- 单任务并发保护

### 依赖

- PostgreSQL 依赖 `pg_dump`
- MySQL 依赖 `mysqldump`

## 服务节点

### 已实现

- 服务启动后自动上报节点心跳
- 节点按类型过滤展示
- 在线 / 离线状态判断
- 展示版本、Git Commit、启动时间、心跳时间、环境、地域、能力标签等

### 节点类型

- `web`
- `worker`
- `scheduler`

## CLI

### 已实现

- `login`、`logout`、`whoami`
- `category` 管理
- `content list`、`content view`
- `content create article`
- `content edit`
- `content delete`
- `comment`、`favorite`、`follow`、`like`
- `profile`
- `user`
- `settings`
- 多 profile 配置隔离
- `en` / `zh-CN` CLI 文案切换

### 当前限制

- `content create` 目前只支持 `article`
- CLI 已识别 `podcast` 类型，但还没有独立的播客/视频/图库发布工作流
