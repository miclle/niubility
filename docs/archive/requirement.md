# Niubility 产品需求文档

## 需求一：内部学习交流平台

合并 Niubility Share / Niubility Training 两个模块，打造一个统一的「内部学习交流」平台。在这里，不仅可以沉淀优质学习成长内容（图文/视频/直播等），还可以让大家发起交流，助力七牛朝着"学习型组织"发展。

> 想象中有点像 B 站的感觉，但也许能有更好的界面呈现方式

### 1. 登录

| 维度 | 说明 |
| --- | --- |
| 入口 | 移动端 & PC 端（而非只有 PC）；可从企业微信「工作台」进入，类似于「同事吧」的感觉，通过 mars 登录一方面操作复杂，另一方面员工活跃度也很低 |
| 账号 | 仅七牛员工有账号与访问权 |

### 2. 内容

#### 2.1 内容类型

- 可发布培训/会议录制视频。例：《AI 实践分享》
- 配套内容：封面页、培训内容标题、培训内容简介、主讲人、主讲人简介

#### 2.2 内容修改

- 发布后仍可修改内容（错别字、补充漏掉的图、删掉某张图等）

#### 2.3 内容检索

| 检索方式 | 示例 |
| --- | --- |
| 标题/内容关键词 | AI 实践分享 |
| 话题标签 | `#AI` |

#### 2.4 内容排序

- 依据发布时间排序
- 依据点赞数排序

### 3. 互动

#### 3.1 评论

- 可发文字、图片、表情包、相关话题的内外部链接
- 可评论他人的评论进行互动
- 管理员可置顶评论

#### 3.2 点赞 & 分享

- 可点赞、收藏内容帖
- 可给评论点赞
- 可将内容转发到企业微信群

### 4. 权限

| 角色 | 能力 |
| --- | --- |
| 管理员 | 发布内容；查询后台数据（点赞人员、下载记录等）；删帖、删评论 |
| 普通用户 | 浏览、评论、点赞、收藏、转发 |

## 需求二：七牛文化平台

将 New Show 从单一的「相册」升级为「七牛文化」的窗口。在这里，七牛人能够看到每一次关键文化内容的报道、精彩文化活动详情（照片/视频/活动成果等），助力七牛文化的沉淀与发酵。

> 较多需求细节与需求一雷同，以下仅标注差异部分
>
> 想象中有点像小红书的感觉，但也许能有更好的界面呈现方式

### 1. 登录

与 [需求一 - 登录](#1-登录) 相同。

### 2. 内容

#### 2.1 内容类型

> **与需求一差异**：支持图文内容，且视频涵盖长/短多种形态

- 可发布长/短图文。例：晨会速报、Hackathon 活动报道
- 可发布长/短视频。例：年会最精彩的 18 张照片、15 秒快剪视频、精彩节目全程回放

#### 2.2 内容修改

- 发布后仍可修改内容（错别字、补充漏掉的图、删掉某张图等）

#### 2.3 内容检索

| 检索方式 | 示例 |
| --- | --- |
| 标题/内容关键词 | 年会、晨会 |
| 话题标签 | `#Hackathon` |

#### 2.4 内容排序

- 依据发布时间排序
- 依据点赞数排序

#### 2.5 内容上传/下载

> **需求一无此功能**

- 管理员可批量上传照片/视频
- 用户可在相应活动帖内下载/获取全部照片/视频
- 解决痛点：年会的几百/几千张照片，因缺少此功能，只能通过百度网盘给到大家

### 3. 互动

#### 3.1 评论

- 可发文字、图片、表情包、相关话题的内外部链接
- 可评论他人的评论进行互动
- 管理员可置顶评论

#### 3.2 点赞 & 分享

- 可给内容帖点赞
- 可给评论点赞
- 可将帖子内容转发到企业微信群

### 4. 权限

| 角色 | 能力 |
| --- | --- |
| 管理员 | 发布内容；查询后台数据（点赞人员、下载记录等）；删帖、删评论 |
| 普通用户 | 浏览、评论、点赞、转发、下载活动资源 |

## 两个需求对比

| 维度 | 需求一：内部学习交流 | 需求二：七牛文化 |
| --- | --- | --- |
| 定位 | 学习成长、知识沉淀 | 文化传播、活动记录 |
| 风格参考 | B 站 | 小红书 |
| 内容形态 | 培训/会议录制视频 | 长短图文 + 长短视频 |
| 批量上传/下载 | - | ✅ |
| 收藏功能 | ✅ | - |
| 其他功能 | 相同（登录、检索、排序、评论、点赞、转发、权限） | 相同 |
