# Niubility 产品功能文档

> 最后更新：2026-03-27

Niubility 是企业内部学习与文化平台，支持视频、图库、文章三种内容类型，提供评论、点赞、收藏、关注等社交功能。

---

## 功能概览

| 模块 | 功能 | 状态 |
|------|------|------|
| 认证与授权 | 登录、注册、SSO | ✅ 已完成 |
| 用户系统 | 个人资料、关注、主页 | ✅ 已完成 |
| 内容管理 | 视频、图库、文章 | ✅ 已完成 |
| 互动功能 | 评论、点赞、收藏 | ✅ 已完成 |
| 搜索与排序 | 关键词、标签、分类 | ✅ 已完成 |
| 管理后台 | 用户、内容、设置管理 | ✅ 已完成 |
| 企业微信 | 部门/用户同步 | ✅ 已完成 |
| 存储系统 | S3 文件上传 | ✅ 已完成 |
| 分享功能 | 转发企业微信群 | 📋 TODO |

## 一、认证与授权

### 1.1 登录方式

| 功能 | 状态 | 说明 |
|------|------|------|
| 密码登录 | ✅ | 用户名 + 密码 |
| OIDC SSO | ✅ | 支持 OIDC 协议（如 Google、Okta） |

### 1.2 角色权限

| user | ✅ | 普通用户，可浏览、评论、点赞、收藏、关注 |

### 1.3 系统初始化

## 技术栈

- **后端**: Go 1.25 + fox (Gin fork) + GORM + PostgreSQL
- **前端**: React 18 + TypeScript + Vite + Tailwind CSS 4 + shadcn/ui
- **认证**: JWT (cookie-based) + SSO (OIDC / SAML 2.0)
- **存储**: S3 兼容对象存储
- **集成**: 企业微信 API
